Редактор исключений из сканирования.


Рис 1. Окно редактора исключений.

Можно добавить в исключения те адреса, которые уже обработались, как успешные. Либо те, которые вы не желаете подвергать обработке программы.

Адреса, которые были занесены в список исключений, будут игнорироваться сканером портов при пробеге диапазонов. При этом сохранится возможность повторного сканирования адреса, который уже есть в таблице. Функция Rescan игнорирует список исключений.

Также список исключений можно изменить, редактируя файл exclusions.txt.

    Содержание

  • 1. Список исключений
  • 2. Кнопки редактирования
  • 3. Кнопки закрытия окна

• Список исключений

Может содержать как отдельные IP адреса, так и адреса вместе с указанными портами.

• Кнопки редактирования

- [Add] - позволяет добавить новый элемент в список исключений.
- [Delete] - удаляет выбранный элемент из списка.
- [Paste from clipboard] - позволяет вставить адреса из буфера обмена (либо удалив предыдущие, либо дописав их в конец списка).

Подсказка: Можно выделить два столбца IP и Port в таблице успешных результатов, скопировать ячейки, и поместить их сюда при помощи вставки.

• Кнопки закрытия окна

Кнопка [OK] сохраняет список исключений и закрывает окно, а кнопка [Cancel] закрывает окно без сохранения изменений.
