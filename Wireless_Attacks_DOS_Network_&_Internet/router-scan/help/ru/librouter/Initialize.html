Функция Initialize

Выполняет инициализацию библиотеки Router Core.

Примечание  Библиотека может быть инициализирована лишь один раз после загрузки, и если инициализация уже была выполнена, функция завершится неудачно.

Синтаксис

Delphi
C++
function Initialize: Bool; stdcall;

BOOL WINAPI Initialize();

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
