Загрузчик 3WiFi.


Рис 1. Окно загрузчика 3WiFi.

Выполняет выгрузку точек доступа на сервер базы данных 3WiFi. Программа сама определяет, сколько точек доступа можно загрузить в данный момент.

    Содержание

  • 1. Область вывода информации
  • 2. Кнопка загрузки
  • 3. Индикаторы состояния
  • 4. Комментарий

• Область вывода информации

Сюда выводится информация о состоянии загрузки данных.

• Кнопка загрузки

Запускет выгрузку данных. В процессе выгрузки позволяет её остановить.

• Индикаторы состояния

Имеется два индикатора - полоса состояния и количество загруженных точек из всех обнаруженных. Изменяются по мере загрузки данных.

• Комментарий

Позволяет задать комментарий для выгружаемых данных.

Рекомендуется указывать комментарий, дающий пояснение об источнике данных, например, название просканированного провайдера.

Пример: Москва, Onlime
