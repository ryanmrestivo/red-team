Процедура WriteLog

Получает подробную информацию из журнала событий обработчика роутеров.

Синтаксис

Delphi
C++
procedure WriteLog(
  Str: PChar;
  Verbosity: Byte
); stdcall;

VOID CALLBACK WriteLog(
  _In_  LPCTSTR Str,
  _In_  BYTE Verbosity
);

Параметры

Str [in]

Строка с сообщением из журнала.

Verbosity [in]

Уровень детализации принятого сообщения.
