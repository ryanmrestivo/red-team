Процедура SetTableData

Получает результаты сканирования от обработчика роутеров.

Синтаксис

Delphi
C++
procedure SetTableData(
  Row: DWORD;
  Name,
  Value: PChar
); stdcall;

VOID CALLBACK SetTableData(
  _In_  DWORD Row,
  _In_  LPCTSTR Name,
  _In_  LPCTSTR Value
);

Параметры

Row [in]

Номер строки таблицы, который был указан при вызове функции PrepareRouter.

Name [in]

Внутреннее имя колонки таблицы.

ЗначениеОписание
stcStatus
"Status"

Описывает текущее состояние обработки.

stcAuth
"Auth"

Содержит информацию об авторизации на устройстве.

stcType
"Type"

Содержит информацию о типе устройства.

stcRadioOff
"RadioOff"

Содержит информацию о состоянии беспроводного устройства.

Возможные значения:
"[X]" - беспроводной интерфейс отключён.
"-" - устройство не имеет беспроводного интерфейса.

stcHidden
"Hidden"

Содержит информацию о трансляторе имени сети.

Возможные значения:
"[X]" - трансляция SSID отключена, точка доступа в невидимом состоянии.
"-" - устройство не имеет беспроводного интерфейса.

stcBSSID
"BSSID"

Содержит BSSID точки доступа (MAC адрес).

Также может содержать строку <no wireless>, которая означает, что устройство не имеет беспроводного интерфейса.

stcSSID
"SSID"

Содержит ESSID точки доступа (имя сети).

stcSec
"Sec"

Содержит тип защиты точки доступа (тип шифрования или аутентификации).

stcKey
"Key"

Содержит парольный ключ к точке доступа.

stcPin
"WPS"

Содержит WPS пин код точки доступа.

stcLANIP
"LANIP"

Содержит локальный IPv4 адрес устройства.

Также может содержать строку <bridge>, которая означает, что интерфейс локальной сети работает в режиме моста или отсутствует.

stcLANMask
"LANMask"

Содержит локальную IPv4 маску сети устройства.

Также может содержать строку <bridge>, которая означает, что интерфейс локальной сети работает в режиме моста или отсутствует.

stcWANIP
"WANIP"

Содержит глобальный IPv4 адрес устройства.

stcWANMask
"WANMask"

Содержит глобальную IPv4 маску сети устройства.

stcWANGate
"WANGate"

Содержит используемый IPv4 шлюз устройства по умолчанию.

stcDNS
"DNS"

Содержит список IPv4 серверов разрешения доменных имён устройства, разделены символом пробела.

 

Value [in]

Значение ячейки таблицы, устанавливаемое обработчиком роутеров.
