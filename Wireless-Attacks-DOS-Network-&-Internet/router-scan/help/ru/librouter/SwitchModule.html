Функция SwitchModule

Включает или отключает выбранный модуль библиотеки Router Core.

Синтаксис

Delphi
C++
function SwitchModule(
  Index: DWORD;
  Enabled: Bool
): Bool; stdcall;

BOOL WINAPI SwitchModule(
  _In_  DWORD Index,
  _In_  BOOL Enabled
);

Параметры

Index [in]

Индекс модуля, начинается с нуля.

Enabled [in]

Включить или отключить выбранный модуль.

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
