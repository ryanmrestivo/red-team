Функция GetModuleCount

Получает количество модулей библиотеки Router Core.

Синтаксис

Delphi
C++
function GetModuleCount(
  var Count: DWORD
): Bool; stdcall;

BOOL WINAPI GetModuleCount(
  _Out_  DWORD *Count
);

Параметры

Count [out]

Количество модулей.

Возвращаемое значение

Если функция выполнилась успешно, возвращается ненулевое значение.

Если же функция выполнилась неудачно, возвратится нуль.
